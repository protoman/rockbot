## Rockbox

An open-source, SDL based, game engine to replicate old style NES games, mostly Megaman ones, and an editor made with Qt, for custom graphics, weapons, bosses, enemies and level creation.

## Requirements

* SDL 1.2
* QT6+

## Build

```shell
./build-ubuntu.sh
```
## Run

```shell
(cd build && ./rockbot)
```
